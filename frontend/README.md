# PricePulse

A full-stack MVP project for tracking products and simulating price changes.

## Tech Stack

- Frontend: React + Vite
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
- Containerization: Docker + Docker Compose

---

## Project Structure

- frontend/ → React UI
- backend/ → Express API
- db → PostgreSQL (Docker service)

---

## How to Run the Project

### 1. Start everything with Docker

```bash
docker compose up --build
